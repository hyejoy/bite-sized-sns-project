```bash
npm i react-router@7
```

# main.tsx

- import BrowserRouter
- App 컴포넌트 감싸기

```tsx
import { BrowserRouter } from "react-router";

createRoot(document.getElementById("root")!).render(
  <BrowserRouter>
    <App />,
  </BrowserRouter>,
);
```

# App.tsx

- Routes - Route 컴포넌트 이용하여 경로 설정해주기

```tsx
import { Route, Routes } from "react-router";
import "./App.css";
export default function App() {
  return (
    <Routes>
      <Route path="/" element={<div>Home</div>} />
      <Route path="/sign-in" element={<div>TEST</div>} />
      <Route path="/sign-up" element={<div>TEST</div>} />
    </Routes>
  );
}
```

# 동일한 레이아웃 갖게 하기

- 동일한 레이아웃을 갖는 경우에는 해당 Route컴포넌트끼리 Route 컴포넌트로 또 묶어주면 됨
- 동일 레이아웃에서 **Outlet컴포넌트**를 명시해줌으로써, 하위 컴포넌트들이 어디에 렌더링되는지 꼭 명시해줘야함

```tsx
import { Outlet, Route, Routes } from "react-router";
import "./App.css";
import IndexPage from "./pages/IndexPage";
import SignInPage from "./pages/SignInPage";
import SignUpPage from "./pages/SignUpPage";

export function AuthLayout() {
  return (
    <div>
      <header>Header!</header>
      <Outlet />
    </div>
  );
}
export default function App() {
  return (
    <>
      <Routes>
        <Route path="/" element={<IndexPage />} />
        <Route element={<AuthLayout />}>
          <Route path="/Signup" element={<SignUpPage />} />
          <Route path="/Signin" element={<SignInPage />} />
        </Route>
      </Routes>
    </>
  );
}
```

- Tailwind CSS Intellisense 확장 프로그램 설치

# Tailwind CSS Intellisense

- tailwind css 유틸리티 클래스 자동완성 제안 기능
- hover: css 적용내용 보여줌

# Tailwind CSS 자동 정렬 기능

- npm i -d prettier prettier-plugin-tailwindcss
- .prettierrc 파일 생성
- 내용 추가하기

```json
{
  "plugins": ["prettier-plugin-tailwindcss"]
}
```

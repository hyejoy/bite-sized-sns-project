# 홈페이지

- [ui/shadcn.com](https://ui.shadcn.com/)

# 설치 과정

- tailwind css 우선 설치 후 진행
- 홈페이지 get start 문서 참고
- https://ui.shadcn.com/docs/installation/vite

```bash
npx shadcn@latest init
```

# 설치 후 변화

- component.json 추가됨 (쉐드시엔 설정 파일)
- 각각의 UI 컴포넌트를 개별 명령어로 설치 진행해야함

# 컴포넌트 설치 방법

```bash
npx shadcn@latest add button

```

# 📱 한입로그 (One Bite Log) - 학습 기록

> **인프런: 한입 크기로 잘라먹는 실전 프로젝트 - SNS편**을 수강하며 배우고 구현하는 과정을 기록합니다.

현대 프론트엔드 생태계의 핵심 기술들을 체득하는 것을 목표로 합니다.

---

## 🛠 학습 기술 스택

이번 프로젝트를 통해 다음 기술들을 마스터할 계획입니다.

- **Framework:** React
- **Styling:** Tailwind CSS, Shadcn/ui
- **State Management:** Zustand (전역 상태 관리)
- **Server State:** TanStack Query (데이터 페칭 & 캐싱)
- **Backend:** Supabase (인증, DB, 스토리지)

---
